# 🚀 8192 Game   

A modern implementation of the classic **8192 puzzle game**, an advanced version of 2048 where players merge tiles strategically to reach the 8192 tile.

---

## 🎮 About the Game

The **8192 Game** is a logic-based puzzle where:

- Players slide tiles in four directions  
- Matching tiles merge into one  
- Goal: reach the **8192 tile**  

👉 It is an extended version of 2048 with higher difficulty and deeper strategy.

## 🧠 Core Logic

- Game board represented as a **2D grid**

### Each move:
- Slides tiles in a direction  
- Merges equal values  
- Adds a new tile (2 or 4)  

### Game ends when:
- No moves left  
**OR**  
- 8192 tile is reached
